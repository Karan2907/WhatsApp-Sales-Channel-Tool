Desktop Application Installation

Step 1: Download

Click the "Download Desktop App" button above to get the latest version of the application.

Step 2: Run Installer

Double-click the installer file to begin the installation process. This self-contained installer includes everything needed to run the application.

Step 3: Launch Application

After installation, launch the application from your desktop shortcut or start menu.

Step 4: Configure

Open your browser and go to http://localhost:3000 to configure your WhatsApp provider settings.

Step 5: Integrate

Configure your booking platform to send webhooks to the provided endpoints.
