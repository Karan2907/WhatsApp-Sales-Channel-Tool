Installation Instructions

Step 1: Download

Click the download button above to get the latest version of the application.

Step 2: Extract

Extract the ZIP file to a folder of your choice on your computer or server.

Step 3: Install Node.js

If you don't have Node.js installed, download and install version 16 or higher from nodejs.org.

Step 4: Run the Application

Double-click the start.bat file or run it from the command line to launch the application.

Step 5: Configure

Open your browser and go to http://localhost:3000 to configure your WhatsApp provider settings.

Step 6: Integrate

Configure your booking platform to send webhooks to the provided endpoints.
