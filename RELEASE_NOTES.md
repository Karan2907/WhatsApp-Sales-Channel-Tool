# WhatsApp Sales Channel Tool - Release Package

## What's Included

1. **WhatsApp-Sales-Channel-Tool-Setup.exe** - Self-contained application package (renamed ZIP file)
   - Includes all application files
   - Requires Node.js v16+ to be installed separately
   - Includes all application dependencies
   - Simple extraction and run installation

2. **Installation Instructions**
   - Simply run the installer
   - Follow the installation wizard
   - The application will be installed with all dependencies
   - Desktop and Start Menu shortcuts will be created automatically

## System Requirements

- Windows 7 or later
- No separate Node.js installation required (included in installer)

## Usage

1. Run the installer
2. Launch the application from desktop shortcut or Start Menu
3. Configure your WhatsApp provider settings
4. Set up webhooks from your booking platform
5. Start engaging with guests through automated WhatsApp flows
